# MAL - 阴历生日倒计时与纪念日记录器

一个优雅的网页应用，用于记录特别日子（如认识纪念日）并支持阴历生日倒计时。🎂🌙

## ✨ 功能亮点

- 同时支持阴历和阳历生日倒计时
- 支持阴历生日到阳历的自动转换
- 智能判断闰月年份，处理同一阴历生日可能出现两次的情况
- 视觉设计精美，响应式布局，适配手机和桌面端
- 动态数字滚动倒计时效果
- 支持返回按钮、人物切换、淡入淡出动画

## 🛠 更新日志

🌟 新增 (New Features)
- 现在可以选择两个人是过阴历生日还是阳历生日，不同的生日习惯不同的倒计时。
✅ 修复 (Fixes)
- 修复时间统一性问题：通过上述网络时间同步功能，修复了因本地时间不准导致倒计时不准确的BUG。
- 修复模式切换失败：修复了在阳历生日用户与阴历生日用户之间切换时，倒计时逻辑和显示模式无法正确更新的BUG。


### v1.1 （2025-06-19）

🌟 新增(New Features)
- 支持自动识别当前年份是否存在 **两个相同阴历生日（如闰六月、六月）**
- 当存在两个生日时，前端提示“今年你有两个生日哦～”，并依次倒计时至第一个、第二个生日
- 当两个都过了，自动进入下一年生日倒计时

✅ 修复(Fixes)
- 修复阴历生日在部分年份无法正确转换为阳历的问题（例如闰月年）


## 🛠 已知BUG

1. 返回按钮动画和样式优化尚未完成，计划后续调整
2. 生日当天没有显示生日祝福，待添加该功能
3. 动画与UI深度优化：当前页面切换动画较为生硬，返回机制不明确。计划将对UI进行深度优化，并采用更专业的动画方案（如GSAP）重构过渡效果。
4. 生日祝福功能：应用在生日当天仅显示倒计时为0，缺少应有的祝福和庆祝环节。
5. 移动端适配：当前的UI在移动设备上的布局和体验尚有不足，需要进行专门的响应式设计与适配。
6. 获取网络时间根据网络情况而定，网络不好则会出现屏幕空档期，修改建议为加入网页启动动画
