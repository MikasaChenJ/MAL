# MAL - 农历生日倒计时与纪念日记录器

一个优雅的网页应用，用于记录特别日子（如认识纪念日）并支持农历生日倒计时。🎂🌙

## ✨ 功能亮点

- 支持农历生日到阳历的自动转换
- 智能判断闰月年份，处理同一农历生日可能出现两次的情况
- 视觉设计精美，响应式布局，适配手机和桌面端
- 动态数字滚动倒计时效果
- 支持返回按钮、人物切换、淡入淡出动画

## 🛠 更新日志

### v1.1 （2025-06-19）

✅ 修复：
- 修复农历生日在部分年份无法正确转换为阳历的问题（例如闰月年）

🌟 新增：
- 支持自动识别当前年份是否存在 **两个相同农历生日（如闰六月、六月）**
- 当存在两个生日时，前端提示“今年你有两个生日哦～”，并依次倒计时至第一个、第二个生日
- 当两个都过了，自动进入下一年生日倒计时
