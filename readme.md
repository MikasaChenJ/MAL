# MAL – 阴历生日倒计时与纪念日记录器

一个优雅的网页应用，用于记录重要日子（如相识纪念日），并支持 **阴历 / 阳历** 生日倒计时。🎂🌙

## ✨ 功能亮点

- **阴历 / 阳历双模式**：一键切换生日计时方式
- **自动阴阳历转换**：基于 lunar-javascript 动态换算，无需手动计算
- **智能闰月处理**：遇到闰月年份自动识别并依次倒数两个生日
- **专属生日祝福**：生日当天自动展示独特的动态彩色祝福语，取代倒计时
- **高度可定制化**：所有纪念日、人员、生日、背景及文案均在独立的配置区修改，无需触碰核心代码
- **统一网络时间**：调用 TimeAPI 保证多端时间一致，倒数更精确
- **每日一言**：每次加载页面时，从一言 API 自动获取一条名言，增添人文气息
- **动画视觉**：数字翻牌、界面淡入淡出及玻璃拟态卡片，交互流畅自然
- **响应式布局**：桌面与移动端均完美适配，提供一致的浏览体验


## 快速开始 (How to Use)

本项目无需复杂的环境配置，只需一个现代浏览器即可运行。

### 步骤 1：获取代码

* **选项A (推荐)**：如果您安装了 Git，请通过 `git clone` 克隆本仓库。
* **选项B**：直接点击本仓库页面上方的 `Code` -> `Download ZIP` 下载全部文件。

### 步骤 2：个性化配置 (核心)

所有自定义内容都集中在 `index.html` 文件顶部的 `<script>` 标签内的 `CONFIG` 对象中。您只需要修改这里即可完成所有配置。

```javascript
const CONFIG = {
  // 1. 主要纪念日倒计时的起始日期
  mainTargetDate: '2025-06-07T02:21:00',

  // 2. 人员信息配置 (可添加多人)
  people: [
    {
      key: 'mikasa',      // 唯一ID (英文，无空格)
      name: 'Mikasa',     // 显示的昵称
      birthMonth: 4,      // 生日月份
      birthDay: 11,       // 生日日期
      isLunar: false       // true代表农历, false代表阳历
    },
    {
      key: 'linlin',
      name: '琳琳',
      birthMonth: 6,
      birthDay: 19,
      isLunar: true
    }
  ],

  // 3. 背景图片
  backgrounds: {
    desktop: 'image/pc.jpg', // 桌面端背景
    mobile: 'image/m.jpg'    // 移动端背景
  },

  // 4. 页脚版权信息
  footer: {
    copyright: '&copy; 2025 By Mikasa & Linlin. All Rights Reserved.'
  },
  
  // 5. API配置 (一般无需修改)
  api: { ... }
};
```

### 步骤 3：替换静态资源

1.  将您自己的**背景图片**放入 `image` 文件夹，并确保文件名与 `CONFIG.backgrounds` 中的配置一致。
2.  替换 `image/logo.png` 为您自己的网站 Logo。

### 步骤 4：完成与访问

在本地直接用浏览器打开 `index.html` 文件即可预览效果。

**注意**：由于项目需要从网络加载字体和API，请确保您的设备已连接到互联网。
